Kokaine Discord Bot

About Kokaine

Kokaine is a multifunctional Discord bot designed to enhance your server's experience. With features like moderation tools, fun commands to make your Discord server management easier and more enjoyable.

Features

  • Modular commands system
  • Moderation tools (kick, ban, mute, etc.)
  • Fun commands (memes, games, etc.)
  • Utility functions (weather, time, etc.)
